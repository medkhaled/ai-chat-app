# Application de Chat IA avec Docker

Application de chat utilisant Ollama comme modèle d'IA local, avec backend Flask, frontend HTML/CSS/JS et base de données PostgreSQL pour stocker l'historique des conversations.

## Prérequis:
    1-Docker et Docker Compose installés
    2-Au moins 4 GB de RAM disponible pour Ollama
    3-Ports 3000, 5000, 5432, et 11434 disponibles

## Fonctionnalités
    ✅ Chat en temps réel avec IA locale
    ✅ Interface moderne et responsive
    ✅ Pas besoin de clé API
    ✅ 100% gratuit et privé
    ✅ Fonctionne hors ligne (après téléchargement du modèle)
    ✅ Sauvegarde automatique de toutes les conversations
    ✅ Historique complet des messages
    ✅ Gestion des conversations (créer, consulter, supprimer)
